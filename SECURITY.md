# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly.

**Do not open a public GitHub issue for security vulnerabilities.**

Instead, please report the vulnerability through GitHub's private security advisory feature:

1. Go to the [Security Advisories page](https://github.com/redhat-actions/chart-verifier/security/advisories)
2. Click **"New draft security advisory"**
3. Fill in the details of the vulnerability

You can also email security reports to the Red Hat Product Security team at <secalert@redhat.com>.

## Supported Versions

| Version | Supported |
| ------- | --------- |
| v1.x    | Yes       |

## Security Measures

This repository has the following security measures enabled:

- **Secret scanning** with push protection
- **Dependabot** for automated dependency updates
- **Least-privilege workflow permissions** (read-only by default)
